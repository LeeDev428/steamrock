# React Native Mobile App (Future Development)

This directory is prepared for the future React Native mobile application for Streamrock Realty Corporation.

## Getting Started

When ready to develop the mobile app, initialize the React Native project here using:

```bash
npx react-native init StreamrockMobile
```

Or for Expo:

```bash
npx create-expo-app StreamrockMobile
```

## Planned Features

- Property browsing and search
- User authentication
- Saved properties/favorites
- Push notifications for new listings
- In-app messaging with agents
- Map integration for property locations
- Virtual tours
